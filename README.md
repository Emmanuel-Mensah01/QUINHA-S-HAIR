# Quinha's Hair Galaxy — Premium Braids & Natural Hair Care Website

Live site: [quinha-s-hair.vercel.app](https://quinha-s-hair.vercel.app)

A premium client pitch/demo website built for **Quinha's Hair Galaxy**, a hair braiding salon and natural hair care brand based in Accra, Ghana (Kokomlemle).

## About the Business

Quinha's Hair Galaxy offers knotless braids, goddess braids, cornrows, natural twists, and other protective styles, alongside a line of natural hair care products (oils, butters, moisturizers). Founded by Laurinda Francesquinha Amorin.

## Features

- Hero section with booking CTA
- About / founder story
- Services overview
- Pricing tiers (GHS)
- Product showcase (hair oils, butter, moisturizer)
- Photo & video gallery of client styles
- Customer testimonials
- FAQ accordion
- Business hours & contact section
- WhatsApp-integrated booking (click-to-chat, pre-filled message)
- Fully responsive design

## Tech Stack

- **Framework:** Next.js 16
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Icons:** Lucide React
- **Deployment:** Vercel
- **Analytics:** Vercel Analytics

## Getting Started (Local Development)

```bash
git clone https://github.com/Emmanuel-Mensah01/QUINHA-S-HAIR.git
cd QUINHA-S-HAIR
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
